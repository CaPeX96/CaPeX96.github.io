数据库起步（菜鸟教程)

  • 指定数据库 use 数据库名称;

  • 创建表

    • create table 表名(
列名1 数据类型1,
列名2 数据类型2,
....
列名n 数据类型n
);

    常用数据类型：

    1. int：整数类型
    2. double:小数类型
    3. date:日期，只包含年月日，yyyy-MM-dd
    4. varchar：字符串

    客户端图形化工具：Navicat

    DML：增删改表中数据

    1. 添加数据：
    2. 语法：
      • insert into 表名(列名1,列名2,...列名n) values(值1,值2,...值n);
    3. 注意：
      1. 列名和值要一一对应。
      2. 如果表名后，不定义列名，则默认给所有列添加值
        • insert into 表名 values(值1,值2,...值n);
      3. 除了数字类型，其他类型需要使用引号(单双都可以)引起来
    4. 删除数据：
    5. 语法：
      • delete from 表名 [where 条件]
    6. 修改数据：
    7. 语法：
      • update 表名 set 列名1 = 值1, 列名2 = 值2,… [where 条件];

    DQL：查询表中的记录

    • select 字段列表
from 表名列表
where 条件列表
group by 分组字段
having 分组之后的条件
order by 排序
limit 分页限定
offset 偏移量

  • 去除重复：

    • distinct

  • 起别名：

    • as：as也可以省略

  • 运算符

    • >、< 、<= 、>= 、= 、<>或!=
    • between…and
    • in( 集合)
    • like：模糊查询
    • 占位符：
    • _:单个任意字符
    • %：多个任意字符
    • is null
    • and 或 &&
    • or 或 ||
    • not 或 !

  • 排序方式：

    • asc：升序，默认的
    • desc：降序

  • 聚合函数：将一列数据作为一个整体，进行纵向的计算。

    • count：计算个数
      • 一般选择非空的列：主键
      • count(*)
    • max：计算最大值
    • min：计算最小值
    • sum：计算和
    • avg：计算平均值

  • 分页查询

    • limit 开始的索引,每页查询的条数;
    • limit 开始的索引 offset 每页查询的条数

  • 多表查询

    • select 字段列表 from 表名1 join 表名2 on 条件
select 字段列表 from 表1 left join 表2 on 条件；
select 字段列表 from 表1 right join 表2 on 条件；
select 字段列表 from 表1,表2 where 表1.id=表2.id；

      ```

  • 子查询：

    • select * from s where sc.score = (select max(score) from sc);
select * from s where id in (select id from s where name like '王__');
select * from s s1 ,(select * from s where s.data >= '2000-01-01') s2
WHERE s1.id = s2.id;
